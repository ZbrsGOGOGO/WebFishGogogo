## 四、V1.5 打磨版（原已知缺口）✅ 已完成

这两项曾是「能看能用」的短板，现已补齐（不在原 spec，作为 V1.5 追加完成）。

### 缺口 1：统一 UI 设计 ✅
- **已做**：新增设计令牌体系 `styles/tokens.css`（CSDN 风格品牌红、间距/圆角/阴影/字体、明暗主题）+ `styles/app-shell.css`；通用组件库 `components/ui/`（Button/Input/Textarea/Card/PageHeader/Tag/Badge/Modal/EmptyState/Spinner）；应用外壳美化（CSDN 风格顶栏导航、登录/注册卡片、首页仪表盘、便签 dock）。
- **方案**：纯 CSS 设计令牌 + CSS Modules，无引入 Tailwind 等重依赖。

### 缺口 2：工具真实交互 ✅
- **已做**：工具运行时框架（registry + ToolRunnerModal，slug→懒加载组件）；12 个工具全部实现真实交互：
  - 时间/计算类：下班倒计时、计时器（含番茄钟）、日期计算、计算器、汇率换算
  - 开发者类：JSON 格式化、时间戳转换、正则测试、单位换算、颜色转换
  - 文本类：文本处理、字数统计
- 点击「启动」在 Modal 中打开真正可操作的工具界面。纯逻辑均有单元 + 属性测试覆盖。

> 前端测试：132 全部通过；typecheck + build 通过。
